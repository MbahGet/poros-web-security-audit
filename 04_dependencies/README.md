# Bab 04 – Analisis Dependensi CTFd

## Files
- `pip_audit_report.json` – Hasil pip-audit (machine-readable)
- `pip_audit_report.txt` – Hasil pip-audit (human-readable)
- `safety_report.txt` – Hasil safety check
- `npm_audit_report.json` – Hasil npm audit (jika ada)
- `dependency_list.txt` – Daftar dependensi Python

## Cara Interpretasi
Setiap temuan mencantumkan: nama package, versi terinstall,
versi fix yang tersedia, dan CVE ID terkait.
