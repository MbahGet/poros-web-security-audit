# Bab 01 – Fondasi Keamanan Web

## Temuan
- Lihat `headers.txt` untuk raw HTTP response headers
- Lihat `security_headers_report.txt` untuk security headers analysis
- Lihat `ssl_report.txt` untuk hasil pemeriksaan TLS/SSL
- Lihat `tls_info.txt` untuk info cipher dan sertifikat

## Tools Used
- `curl` – HTTP header inspection
- `testssl.sh` – TLS/SSL analysis
